born of ens

ENS (Ethereum Name Service) was launched on 4th May 2017. About 12% of the names were possible to be registered through an auction with certain complexities to protect the bidders and their names. Almost all names including emojis, Non-ASCII characters and more, were allowed to be registered as long as their ​character length was equal or more than 7.

*First version of ENS logo

Limited supply

Pre-Punk Category consists of 79720 domains. These names are ranked based on their "Created At" date in their metadata making it the only collection with a rarity factor.

Initiated by an ENS Dev

Makoto Inoue, an ENS developer, was the first person to release a query at Dune named OG ENS; stating the "Pre-Punk" status of each name. 

Subcategories

What makes "Pre-Punk" so special, is the variety of different categories of names like different languages, brand names, first names, OG digits and many more!

Different projects, One Goal

Projects and collections like namecoins, .bit and a couple of others initiated the concept of using a 'name' interchangabely with their ethereum wallet address (0x...).

ENS, being omni-chain succeded in gaining the most adoption with over 2.8 Million domains registered as of writing this and now is being used as the primary Name Service on many blockchains.

FAQ

You can find the most frequently asked questions below, if you did not find your answer below; join our discord and the team will assist your questions/concerns.

How can I check if my ENS domain is a Pre-Punk? 

There are different ways to confirm if you own a Pre-Punk domain, the easiest being using the original list of Prepunk names to type in your name at the bottom and check it's status. The "registered date" has to be ranging between 9th May, 2017 to 23rd June, 2017.

NOTE: You need to use the modified query/search hash incase your name is a 'Lost' one in the blank spaces. (more info below)

What are the subcategories under Pre-Punk?

There different main subcategories/filters currently including:

  • OG Digits:  The Pre-Punk ENS domains that contain only digits.
  • Top-1k / Top-10k: This term refers to the first 1000/10000 domains ever registered through ENS registrar.
  • Ethmoji: These Pre-Punks are the first Domains to carry an emoji.
  • Palindromes

This categories/filters can be updated as adoption grows.

What are the Blank Spaces in the Makoto Inu Dune's query?

During auction, The names that were not known (not included in the initial database) were hashed and were hidden to protect the bidder. During the ENS smart contract migration a lot of these names, due to different reasons which includes but not limited to using "not allowed" charachters, were lost and only their "keccak256" hash was transferred. This type of hashing is only used for encrypting therefore is one-way. 

Us, at Pre-Punk Club used our resources to 'crack' these hashes and recover over 90% of the names lost in the Top1k. If yo would like to check these found names, There is a modified version of Makoto's dashboard that includes all blanks that have been discovered and registered. Any new names that are found going forward will be automatically included.

Where can I trade Pre-Punk domains?

Currently, The generic marketplaces like OpenSea, LooksRare and X2Y2, do not provide the utility/filter based on their creation date which can make tracking these invaluable names harder.

However, Thanks to the team at ens.vision, you can use this ENS-Focused marketplace to find Pre-Punk category and to use the filter "Creation date Asc", to list all the known and available Pre-Punk names based on their created date/rank.

UPDATE: The subcategories of TOP1K and TOP10K are now aggregated at ENS.CLUB.

You can also join our Discord to use related channels for trading.

Are ENS domains with Coloured background (Not the standard Blue), Prepunks?

No, Before ENS set the standard blue avatar, Opensea created different coloured backgrounds for each name. 

Using the background colour to tell if a name is Pre-Punk is almost always not-accurate. At the moment The background for all ENS names including Pre-Punks is Standard Blue. If you encounter a name with a different coloured background, It is because it's metadata on that certain marketplace is not refreshed.

ALWAYS use "Created At" date in the metadata or NFT details and/or use Makoto Inu query to confirm a Pre-Punk.

What are "Chaff" names? Are they Pre-Punks?

No, Chaff names are the names that were used along the "actual" name being bidded on to "cover" and protect the bidder. Chaff names are all also in in the preimages.js file in the resources.

These names "may" report the created date within prepunk dates, However, They lack the finalized auction transaction and are not available in the OG ENS Dune query; making them a non-prepunk domain.

When is the "Cut-off" for Pre-Punk names?

Below you can find important dates with their epoch* format.

9/May/2017 (1494328939000): Start of PrePunk status

13/May/2017 (1494656431000): End of Top 1000

20/May/2017 (1495320350000): End of Top 10,000

23/June/2017 (1498252062000): End of PrePunk status


*In a computing context, an epoch is the date and time relative to which a computer's clock and timestamp values are determined. This is the format used in ENS metadata to report these dates with exact minute/second and if converted by the marketplace; it will show the "human-readable" format of time.

You can use epochconverter.com to convert these times by yourself.


I have a PrePunk but the name is not shown in OG ENS Dune Query (Makoto). Why?

The original Makoto's query is not updated since release therefore it will not show any PrePunk name from blank spaces that was found.

You can use the Modified Query available in resources section, this query is self-updating all the names that are found.

What is the difference between OG ENS and Pre-Punk? Are they the same?

Considering this trend was initiated by Makoto Inoue, ENS dev, we will explain his vision and the difference of these two names according to his query:

OG ENS:  Registered within 2017-May-09  to 2020-August-27 (274098 items)

Pre-Punk Domains: Registered within 2017-May-09  to 2017-June-23  (79720 items)

So while every Pre-Punk is a OG ENS, Not every OG ENS is a Pre-Punk.

How can I find Blank Spaces?

Every string/name can be converted to (keccak256) hash, but the otherway around is not possible. Current methods used to "uncover" these blanks are; Bruteforcing, Database lookups, Dictionary attack, Blockchain Txs, and hand-digging the web! Creativity and patience is a key factor in your success!

If you are about to start hunting blank spaces, we strongly suggest you join our official discord channel and use the tips and different workarounds shared by our fellow Blank Busters!
